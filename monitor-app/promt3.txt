Развернута информационная система (ИС)
Компоненты ИС
- nginx в ДМЗ (несколько развнозначных серверов)
- сервера приложений (AS) (несколько развнозначных серверов) 
- Микросервисы ( запущены по одному экземпляру на каждом AS)
- nginx, который маршрутизирует запрос на мискросервисы в рамках AS (запущены на каждом AS)
- кластер серверов с redis
- кластер серверов с rabbitmq
- кластер серверов с kafka + zookeeper
- кластер серверов с postgres
- кластер серверов с keycloak
- кластер серверов с consul
- кластер серверов с ceph
- кластер серверов с opensearch

Пользователи получают доступ к ИС через интернет
Сотрудники получают доступ через внутренюю сеть

Развернут prometheus 2.31 и grafana 11

Необходимо построить дашборд в grafana, котоый будет показывать общий статус ИС и состояние всех компонет.

На дашборде должно быть по одной stats панели для каждого типа компонента. Такая панель отображает совокупный статус компонента.

Статуса три
- ok - все экземпляры компонента работает без замечаний
- warning - в работе компонента есть нефатальные ошибки. Например из нескольких nginx в ДМЗ недоступен один
- error - компонент не работает 

Для каждого компонента укажи какие метрики и спомощью чего следует собирать и как по этим метрикам вычислять совокупный статус компонента
