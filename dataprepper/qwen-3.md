Вот готовый **Grafana Dashboard JSON** для визуализации метрик Data-Prepper. Дашборд структурирован по основным компонентам: **Armeria (сервер/executor)**, **Peer Forwarder**, **Entry Pipeline**, **Raw Pipeline**, **Service Map Pipeline**, и **Общие метрики (CPU, процесс)**.

---

✅ **Особенности дашборда:**
- Группировка по логическим блокам.
- Использование `Prometheus` как источника данных.
- Включены ключевые метрики: latency, throughput, errors, buffer usage, executor stats.
- Поддержка alert-индикаторов (можно добавить позже).
- Оптимизирован под Data-Prepper 2.x+ с OpenTelemetry и OpenSearch.

### 💡 Рекомендации:

- Для продакшена добавьте **Alert Rules** на:
  - `core_peerForwarder_badRequests_total > 0`
  - `raw_pipeline_opensearch_documentErrors_total > 0`
  - `armeria_executor_queued_tasks > 1000`
  - `process_cpu_usage > 90%`
- Можно добавить **variable** для фильтрации по `instance` или `job`, если у вас несколько Data-Prepper инстансов.
- Для глубокого анализа — используйте `Explore` с запросами к `armeria_server_connections_lifespan_seconds_*`.
