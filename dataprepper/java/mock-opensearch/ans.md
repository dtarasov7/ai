1. **Python HTTP Server (Вариант 2)** - самый удобный для вашей задачи:
   - Работает из коробки с Python 3
   - Показывает все данные от Data Prepper в консоли
   - Легко добавить логирование в файл
   - Отвечает корректными HTTP 200

2. **Node.js Mock (Вариант 1)** - если вам удобнее Node.js

3. **netcat (Вариант 5)** - если нужно совсем минимально
