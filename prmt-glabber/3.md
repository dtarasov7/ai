Ниже — полный MVP-комплект под схему:

```text
Alertmanager → bridge
Glaber main → Glaber proxy → bridge
```

То есть **Alertmanager пушит алерты в bridge**, а **Glaber через proxy сам опрашивает bridge**, получает LLD JSON и автоматически создаёт item/trigger prototypes. Это соответствует вашему требованию: items в Glaber создаются из результата запроса `Glaber → Glaber proxy → bridge`.

Alertmanager поддерживает `webhook_configs` и `send_resolved`, а HTTP-agent checks в Glaber/Zabbix-подобной модели выполняются proxy, если host monitored by proxy. ([prometheus.io][1]) LLD создаёт items/triggers автоматически по данным discovery. ([Zabbix][2])

---

# 1. Итоговая схема

```text
work contour

Prometheus
   │
   ▼
Alertmanager
   │ POST /alertmanager/v1/webhook
   ▼
am-glaber-bridge
   │
   │ GET /glaber/v1/lld
   │ GET /glaber/v1/alert/<id>/status
   │ GET /glaber/v1/alert/<id>/summary
   │
   ▲
Glaber proxy
   ▲
   │
Glaber main

main contour
```

Bridge хранит состояние алертов в SQLite. Поэтому Glaber может опрашивать bridge как обычный HTTP endpoint, а не принимать входящие соединения из Alertmanager напрямую.

# 4. Python 3 bridge

Файл:

```text
/usr/local/bin/am-glaber-bridge.py
```

Код без Flask/FastAPI, только стандартная библиотека Python 3.

```python
#!/usr/bin/env python3

import argparse
import datetime as dt
import hashlib
import hmac
import json
import os
import sqlite3
import sys
import threading
import time
import urllib.parse
from http.server import BaseHTTPRequestHandler, ThreadingHTTPServer

try:
    import yaml
except ImportError:
    yaml = None


DEFAULT_CONFIG = {
    "listen_host": "0.0.0.0",
    "listen_port": 8080,
    "auth": {
        "bearer_token": "",
    },
    "storage": {
        "sqlite_path": "/var/lib/am-glaber-bridge/alerts.sqlite3",
    },
    "alert_retention": {
        "keep_resolved_seconds": 604800,
        "keep_never_seen_seconds": 86400,
    },
    "glaber": {
        "lld_include_resolved": True,
    },
    "labels": {
        "instance_label": "instance",
        "severity_label": "severity",
        "alertname_label": "alertname",
    },
    "severity_map": {
        "disaster": 5,
        "critical": 4,
        "high": 4,
        "average": 3,
        "warning": 2,
        "info": 1,
        "information": 1,
        "none": 0,
    },
}


def deep_merge(a, b):
    result = dict(a)
    for k, v in b.items():
        if isinstance(v, dict) and isinstance(result.get(k), dict):
            result[k] = deep_merge(result[k], v)
        else:
            result[k] = v
    return result


def load_config(path):
    if not path:
        return DEFAULT_CONFIG

    if yaml is None:
        raise RuntimeError(
            "PyYAML is required for YAML config. Install: python3 -m pip install pyyaml"
        )

    with open(path, "r", encoding="utf-8") as f:
        user_cfg = yaml.safe_load(f) or {}

    return deep_merge(DEFAULT_CONFIG, user_cfg)


def now_utc_iso():
    return dt.datetime.now(dt.timezone.utc).isoformat()


def parse_time_to_epoch(value):
    if not value:
        return None

    try:
        value = value.replace("Z", "+00:00")
        return int(dt.datetime.fromisoformat(value).timestamp())
    except Exception:
        return None


def safe_macro_value(value, default="unknown"):
    if value is None:
        value = default

    value = str(value).strip()
    if not value:
        value = default

    # Для LLD macros и item keys лучше не тащить кавычки/переводы строк.
    value = value.replace("\n", " ").replace("\r", " ")
    value = value.replace('"', "'")
    return value[:255]


def stable_fingerprint(labels):
    """
    Alertmanager обычно присылает fingerprint.
    Если его нет, считаем стабильный hash по labels.
    """
    normalized = json.dumps(labels or {}, sort_keys=True, ensure_ascii=False)
    return hashlib.sha256(normalized.encode("utf-8")).hexdigest()[:16]


class AlertStore:
    def __init__(self, sqlite_path, cfg):
        self.sqlite_path = sqlite_path
        self.cfg = cfg
        self.lock = threading.Lock()

        os.makedirs(os.path.dirname(sqlite_path), exist_ok=True)
        self._init_db()

    def _connect(self):
        return sqlite3.connect(self.sqlite_path, timeout=30)

    def _init_db(self):
        with self._connect() as conn:
            conn.execute(
                """
                CREATE TABLE IF NOT EXISTS alerts (
                    fingerprint TEXT PRIMARY KEY,
                    status TEXT NOT NULL,
                    alertname TEXT NOT NULL,
                    instance TEXT NOT NULL,
                    severity TEXT NOT NULL,
                    severity_num INTEGER NOT NULL,
                    summary TEXT NOT NULL,
                    description TEXT NOT NULL,
                    labels_json TEXT NOT NULL,
                    annotations_json TEXT NOT NULL,
                    starts_at TEXT,
                    ends_at TEXT,
                    generator_url TEXT,
                    updated_at_epoch INTEGER NOT NULL,
                    updated_at_iso TEXT NOT NULL
                )
                """
            )

            conn.execute(
                """
                CREATE INDEX IF NOT EXISTS idx_alerts_status
                ON alerts(status)
                """
            )

            conn.execute(
                """
                CREATE INDEX IF NOT EXISTS idx_alerts_updated_at
                ON alerts(updated_at_epoch)
                """
            )

    def upsert_alert(self, alert):
        labels = alert.get("labels") or {}
        annotations = alert.get("annotations") or {}

        label_cfg = self.cfg["labels"]
        severity_map = self.cfg["severity_map"]

        fp = alert.get("fingerprint") or stable_fingerprint(labels)

        alertname = safe_macro_value(
            labels.get(label_cfg["alertname_label"]) or labels.get("alertname")
        )
        instance = safe_macro_value(
            labels.get(label_cfg["instance_label"]) or labels.get("instance")
        )
        severity = safe_macro_value(
            labels.get(label_cfg["severity_label"]) or labels.get("severity") or "none"
        ).lower()

        severity_num = int(severity_map.get(severity, 0))

        summary = safe_macro_value(
            annotations.get("summary")
            or annotations.get("message")
            or annotations.get("description")
            or alertname,
            default=alertname,
        )

        description = safe_macro_value(
            annotations.get("description")
            or annotations.get("runbook")
            or annotations.get("summary")
            or "",
            default="",
        )

        status = alert.get("status") or "unknown"
        starts_at = alert.get("startsAt")
        ends_at = alert.get("endsAt")
        generator_url = alert.get("generatorURL") or ""

        updated_epoch = int(time.time())
        updated_iso = now_utc_iso()

        with self.lock:
            with self._connect() as conn:
                conn.execute(
                    """
                    INSERT INTO alerts (
                        fingerprint,
                        status,
                        alertname,
                        instance,
                        severity,
                        severity_num,
                        summary,
                        description,
                        labels_json,
                        annotations_json,
                        starts_at,
                        ends_at,
                        generator_url,
                        updated_at_epoch,
                        updated_at_iso
                    )
                    VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
                    ON CONFLICT(fingerprint) DO UPDATE SET
                        status = excluded.status,
                        alertname = excluded.alertname,
                        instance = excluded.instance,
                        severity = excluded.severity,
                        severity_num = excluded.severity_num,
                        summary = excluded.summary,
                        description = excluded.description,
                        labels_json = excluded.labels_json,
                        annotations_json = excluded.annotations_json,
                        starts_at = excluded.starts_at,
                        ends_at = excluded.ends_at,
                        generator_url = excluded.generator_url,
                        updated_at_epoch = excluded.updated_at_epoch,
                        updated_at_iso = excluded.updated_at_iso
                    """,
                    (
                        fp,
                        status,
                        alertname,
                        instance,
                        severity,
                        severity_num,
                        summary,
                        description,
                        json.dumps(labels, ensure_ascii=False, sort_keys=True),
                        json.dumps(annotations, ensure_ascii=False, sort_keys=True),
                        starts_at,
                        ends_at,
                        generator_url,
                        updated_epoch,
                        updated_iso,
                    ),
                )

        return fp

    def cleanup(self):
        keep_resolved = int(self.cfg["alert_retention"]["keep_resolved_seconds"])
        keep_never_seen = int(self.cfg["alert_retention"]["keep_never_seen_seconds"])
        now_epoch = int(time.time())

        resolved_before = now_epoch - keep_resolved
        stale_before = now_epoch - keep_never_seen

        with self.lock:
            with self._connect() as conn:
                conn.execute(
                    """
                    DELETE FROM alerts
                    WHERE status = 'resolved'
                      AND updated_at_epoch < ?
                    """,
                    (resolved_before,),
                )

                conn.execute(
                    """
                    DELETE FROM alerts
                    WHERE status NOT IN ('firing', 'resolved')
                      AND updated_at_epoch < ?
                    """,
                    (stale_before,),
                )

    def get_all_for_lld(self):
        include_resolved = bool(self.cfg["glaber"]["lld_include_resolved"])

        with self.lock:
            with self._connect() as conn:
                conn.row_factory = sqlite3.Row

                if include_resolved:
                    rows = conn.execute(
                        """
                        SELECT *
                        FROM alerts
                        ORDER BY severity_num DESC, alertname, instance
                        """
                    ).fetchall()
                else:
                    rows = conn.execute(
                        """
                        SELECT *
                        FROM alerts
                        WHERE status = 'firing'
                        ORDER BY severity_num DESC, alertname, instance
                        """
                    ).fetchall()

        return [dict(r) for r in rows]

    def get_alert(self, fingerprint):
        with self.lock:
            with self._connect() as conn:
                conn.row_factory = sqlite3.Row
                row = conn.execute(
                    """
                    SELECT *
                    FROM alerts
                    WHERE fingerprint = ?
                    """,
                    (fingerprint,),
                ).fetchone()

        return dict(row) if row else None

    def stats(self):
        with self.lock:
            with self._connect() as conn:
                conn.row_factory = sqlite3.Row

                total = conn.execute(
                    "SELECT COUNT(*) AS c FROM alerts"
                ).fetchone()["c"]

                firing = conn.execute(
                    "SELECT COUNT(*) AS c FROM alerts WHERE status = 'firing'"
                ).fetchone()["c"]

                resolved = conn.execute(
                    "SELECT COUNT(*) AS c FROM alerts WHERE status = 'resolved'"
                ).fetchone()["c"]

        return {
            "total": total,
            "firing": firing,
            "resolved": resolved,
            "time": now_utc_iso(),
        }


class BridgeHandler(BaseHTTPRequestHandler):
    server_version = "am-glaber-bridge/1.0"

    def log_message(self, fmt, *args):
        sys.stderr.write(
            "%s %s\n" % (self.log_date_time_string(), fmt % args)
        )

    def _send_json(self, code, obj):
        body = json.dumps(obj, ensure_ascii=False, sort_keys=True).encode("utf-8")
        self.send_response(code)
        self.send_header("Content-Type", "application/json; charset=utf-8")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def _send_text(self, code, value):
        body = str(value).encode("utf-8")
        self.send_response(code)
        self.send_header("Content-Type", "text/plain; charset=utf-8")
        self.send_header("Content-Length", str(len(body)))
        self.end_headers()
        self.wfile.write(body)

    def _auth_ok(self):
        token = self.server.cfg["auth"].get("bearer_token") or ""
        if not token:
            return True

        header = self.headers.get("Authorization", "")
        expected = "Bearer " + token

        return hmac.compare_digest(header, expected)

    def _read_json_body(self):
        length = int(self.headers.get("Content-Length", "0"))
        raw = self.rfile.read(length)
        if not raw:
            return {}

        return json.loads(raw.decode("utf-8"))

    def do_POST(self):
        parsed = urllib.parse.urlparse(self.path)

        if parsed.path != "/alertmanager/v1/webhook":
            self._send_json(404, {"error": "not found"})
            return

        if not self._auth_ok():
            self._send_json(401, {"error": "unauthorized"})
            return

        try:
            payload = self._read_json_body()
        except Exception as e:
            self._send_json(400, {"error": "invalid json", "details": str(e)})
            return

        alerts = payload.get("alerts") or []
        saved = []

        for alert in alerts:
            try:
                fp = self.server.store.upsert_alert(alert)
                saved.append(fp)
            except Exception as e:
                self.log_message("failed to save alert: %s", e)

        self.server.store.cleanup()

        self._send_json(
            200,
            {
                "status": "ok",
                "received": len(alerts),
                "saved": len(saved),
                "fingerprints": saved,
            },
        )

    def do_GET(self):
        parsed = urllib.parse.urlparse(self.path)
        path = parsed.path.rstrip("/")

        if path == "/health":
            self._send_json(200, {"status": "ok", "time": now_utc_iso()})
            return

        if path == "/glaber/v1/stats":
            self._send_json(200, self.server.store.stats())
            return

        if path == "/glaber/v1/lld":
            self._handle_lld()
            return

        prefix = "/glaber/v1/alert/"
        if path.startswith(prefix):
            rest = path[len(prefix):]
            parts = rest.split("/")

            if len(parts) != 2:
                self._send_json(404, {"error": "invalid alert endpoint"})
                return

            fingerprint, field = parts
            self._handle_alert_field(fingerprint, field)
            return

        self._send_json(404, {"error": "not found"})

    def _handle_lld(self):
        rows = self.server.store.get_all_for_lld()

        data = []
        for r in rows:
            labels = json.loads(r["labels_json"])

            data.append(
                {
                    "{#ALERT_ID}": r["fingerprint"],
                    "{#ALERTNAME}": r["alertname"],
                    "{#INSTANCE}": r["instance"],
                    "{#SEVERITY}": r["severity"],
                    "{#SEVERITY_NUM}": str(r["severity_num"]),
                    "{#STATUS}": r["status"],
                    "{#SUMMARY}": r["summary"],
                    "{#JOB}": safe_macro_value(labels.get("job", "unknown")),
                }
            )

        # Классический формат LLD для Zabbix/Glaber.
        self._send_json(200, {"data": data})

    def _handle_alert_field(self, fingerprint, field):
        alert = self.server.store.get_alert(fingerprint)

        if alert is None:
            # Если item уже создан, а alert удалён по retention,
            # возвращаем безопасное состояние.
            if field == "status":
                self._send_text(200, "0")
            elif field == "severity_num":
                self._send_text(200, "0")
            else:
                self._send_text(200, "")
            return

        if field == "status":
            self._send_text(200, "1" if alert["status"] == "firing" else "0")
            return

        if field == "status_text":
            self._send_text(200, alert["status"])
            return

        if field == "severity_num":
            self._send_text(200, str(alert["severity_num"]))
            return

        if field == "summary":
            self._send_text(200, alert["summary"])
            return

        if field == "description":
            self._send_text(200, alert["description"])
            return

        if field == "labels":
            self._send_json(200, json.loads(alert["labels_json"]))
            return

        if field == "annotations":
            self._send_json(200, json.loads(alert["annotations_json"]))
            return

        if field == "raw":
            self._send_json(200, alert)
            return

        self._send_json(404, {"error": "unknown field"})


def main():
    parser = argparse.ArgumentParser()
    parser.add_argument(
        "--config",
        default="/etc/am-glaber-bridge/config.yaml",
        help="Path to config.yaml",
    )
    args = parser.parse_args()

    cfg = load_config(args.config)
    store = AlertStore(cfg["storage"]["sqlite_path"], cfg)

    server = ThreadingHTTPServer(
        (cfg["listen_host"], int(cfg["listen_port"])),
        BridgeHandler,
    )

    server.cfg = cfg
    server.store = store

    print(
        f"am-glaber-bridge listening on "
        f"{cfg['listen_host']}:{cfg['listen_port']}",
        flush=True,
    )

    server.serve_forever()


if __name__ == "__main__":
    main()
```

# 6. Glaber: host и template

## 6.1 Host

Создайте host, например:

```text
Host name: prometheus-alerts-work
Visible name: Prometheus alerts - work contour
Groups: Monitoring/Prometheus
Monitored by proxy: glaber-proxy-work
Templates: Template App Prometheus Alertmanager Bridge
```

Макросы host/template:

```text
{$AM_BRIDGE_URL} = http://am-glaber-bridge.work.local:8080
{$AM_BRIDGE_UPDATE} = 30s
```

Важно: host должен быть назначен на `glaber-proxy-work`. Тогда HTTP-agent запросы к bridge будет выполнять именно proxy в контуре `work`.

# 7. Glaber template: логическая конфигурация

Ниже — полная структура шаблона. В UI Glaber/Zabbix она создаётся как template с HTTP-agent LLD rule и item prototypes.

## Template

```yaml
template:
  name: "Template App Prometheus Alertmanager Bridge"
  groups:
    - "Templates/Applications"

  macros:
    - macro: "{$AM_BRIDGE_URL}"
      value: "http://am-glaber-bridge.work.local:8080"

    - macro: "{$AM_BRIDGE_UPDATE}"
      value: "30s"
```

---

## 7.1 LLD rule

```yaml
discovery_rule:
  name: "Prometheus Alertmanager alerts discovery"
  type: "HTTP agent"
  key: "am.alerts.discovery"
  url: "{$AM_BRIDGE_URL}/glaber/v1/lld"
  request_method: "GET"
  update_interval: "{$AM_BRIDGE_UPDATE}"
  timeout: "10s"
  required_status_codes: "200"
  value_type: "Text"

  lld_macros:
    - macro: "{#ALERT_ID}"
      path: "$.data[*].{#ALERT_ID}"

    - macro: "{#ALERTNAME}"
      path: "$.data[*].{#ALERTNAME}"

    - macro: "{#INSTANCE}"
      path: "$.data[*].{#INSTANCE}"

    - macro: "{#SEVERITY}"
      path: "$.data[*].{#SEVERITY}"

    - macro: "{#SEVERITY_NUM}"
      path: "$.data[*].{#SEVERITY_NUM}"

    - macro: "{#SUMMARY}"
      path: "$.data[*].{#SUMMARY}"

    - macro: "{#JOB}"
      path: "$.data[*].{#JOB}"
```

Если ваша версия Glaber принимает классический LLD JSON напрямую, то отдельное описание `lld_macros` может не понадобиться, потому что bridge уже отдаёт:

```json
{
  "data": [
    {
      "{#ALERT_ID}": "abc123",
      "{#ALERTNAME}": "NodeDown",
      "{#INSTANCE}": "srv01:9100",
      "{#SEVERITY}": "critical"
    }
  ]
}
```

# 9. Важная настройка lifetime discovered resources

В LLD rule в Glaber задайте:

```text
Keep lost resources period: 7d
```

И в bridge:

```yaml
alert_retention:
  keep_resolved_seconds: 604800
```

Так resolved alert не исчезнет мгновенно из discovery, а Glaber успеет получить `status=0` и закрыть trigger.

# 10. Минимальный firewall

В контуре `work` достаточно:

```text
Alertmanager → am-glaber-bridge:8080
Glaber proxy → am-glaber-bridge:8080
Glaber proxy → Glaber main
```

Не нужно открывать из `main` доступ к Prometheus или Alertmanager.
