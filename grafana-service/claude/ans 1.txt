Для создания такой панели в Grafana нужно использовать PromQL запрос, который найдет разность между списком сервисов из переменной `srv` и сервисами, которые отдают метрику `uptime`.

Вот решение для панели типа Table:Основная идея решения заключается в использовании оператора `unless` в PromQL, который позволяет найти разность между двумя множествами метрик.

Ключевые моменты:

1. **Основной запрос**: Создаем искусственный вектор со всеми сервисами из переменной `$srv`, а затем исключаем из него те сервисы, которые присутствуют в метрике `uptime`.

2. **Переменная $srv**: Должна содержать список всех сервисов, которые мы хотим мониторить. Обычно получается из `node_systemd_unit_state`.

3. **Результат**: Панель покажет только те сервисы, которые есть в списке `srv`, но не отдают метрику `uptime`.
