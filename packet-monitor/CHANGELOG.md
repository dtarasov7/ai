# Changelog

All notable changes to this project will be documented in this file.

## [1.6.0] - 2026-02-28

### Added

- First public release.
- Terminal UI packet monitor built with Python, Scapy, and Urwid.
- Live capture from a selected interface or `any`.
- Offline analysis of `pcap` and `pcapng` files.
- Packet list and flows/conversations views.
- Packet details with parsed fields and hexdump.
- UI filtering, payload search, exclusions, and profiles.
- Follow Stream for TCP reassembly and TLS handshake summaries.
- Packet export for all captured packets or only filtered packets.
- Extended IPv6 statistics option.
- User guides in English and Russian.
- Architecture diagrams in PlantUML format.
