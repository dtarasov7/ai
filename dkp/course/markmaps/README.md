# Markmap-карты курса

В этой папке лежат markdown-файлы, из которых можно строить интерактивные карты курса через `markmap`.

Как использовать:
- открывать файлы как обычный Markdown;
- рендерить их через локальный `markmap`-инструмент;
- использовать как основу для презентаций, визуальных шпаргалок и быстрого повторения перед лабораторными.

Состав:
- `chapter_01_foundations.mm` — глава 1, общая карта security-слоёв Kubernetes и DKP;
- `chapter_02_pod_security.mm` — глава 2, Pod security и политики допуска;
- `chapter_03_network_mtls.mm` — глава 3, сеть, default deny и mTLS;
- `chapter_04_access_management.mm` — глава 4, IAM и role model;
- `chapter_05_argocd_access.mm` — глава 5, Argo CD, `operator-argo`, ограниченный доступ и GitOps-риски;
- `chapter_06_certs_and_secrets.mm` — глава 6, сертификаты, секреты и delivery patterns;
- `chapter_07_scanning_and_audit.mm` — глава 7, supply chain, scanning, runtime audit и triage;
- `chapter_08_multitenancy.mm` — глава 8, мультиарендность, `Project` и `ProjectTemplate`;
- `chapter_09_compliance_cse.mm` — глава 9, compliance, CIS и DKP CSE;
- `chapter_10_neuvector_luntry.mm` — глава 10, NeuVector, Luntry и внешние security-платформы;
- `chapter_11_module_risk_matrix.mm` — глава 11, оценка модулей Deckhouse как изменения модели угроз.
