# Листы лабораторных работ

В этой папке лежат отдельные листы лабораторных для одиннадцати глав курса.

Каждый лист содержит:
- цель и ожидаемый результат;
- входные артефакты;
- последовательность действий;
- список сдаваемых материалов;
- критерии завершения.
