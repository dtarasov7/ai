# Дашборд для отслеживания версий ПО

Я подготовил полный JSON дашборд с двумя панелями для решения ваших задач.

## Предполагаемая структура метрики

Метрика должна быть типа `gauge` со значением `1`:
```
app{nodename="server-01", appname="nginx", version="1.20.1"} 1
app{nodename="server-01", appname="postgres", version="14.5"} 1
```

## Описание панелей

### 1. Таблица изменений версий
- Показывает текущую и предыдущую версию
- Отображает время последнего изменения
- Использует агрегацию с `changes()` для определения моментов смены версии

### 2. График истории версий (Time series с текстовыми метками)
- Показывает временную шкалу изменений версий
- Каждая версия отображается как отдельный горизонтальный ряд
- Включает фильтры по nodename и appname
